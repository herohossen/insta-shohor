Q: How does javascript work?

Ans: Javascript is a clint-side scripting language. It only workes inside the browser. The browser engine helps javascript codes to run. When JavaScript code is written into an HTML page and a user requests an HTML page with JavaScript in it, the script is sent to the browser and the browser will decide what to do with it. JavaScript language uses DOM to have access any model or notion of web pages, HTML documents, SVG documents, and their component parts. Javascript can also run in computer by node.js

Q: What is the difference between local storage and session storage?

Ans:
S/N LocalStorage SessionStorage
1 Local storage can store 5mb data. Session storage can store 5mb-10mb data
2 The data does not expire. It remains after the browser restart and even OS reboot. It expire when the current browser tab is closed.
3 Changes made are saved and available for future uses. Changes are only available per tab.
4 Used for storing data for long time Used for storing data for short time
5 It has no default expiry Data delete automatically after closing a tab.
